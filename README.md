# Edgar Allen Poe-it

## Description

In this application a user can search for and view poems for inspiration.
They can then compose and save their own written works.
The user must log in in order to compose and save.


## Technologies Used

-Bootstrap
-PoetryDB
-Node
-Express
-MySQL
-Sequelize
-Heroku
